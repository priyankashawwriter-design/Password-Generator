# 🔐 Python Password Generator

A simple and secure Python password generator that creates random 12-character passwords using uppercase letters, lowercase letters, digits, and special characters.

## Features

- Generate secure passwords
- 12 characters long
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters

## Requirements

Python 3.x

## Run

```bash
python password_generator.py
```

Example Output

```
Secure Password: X7@qLm9#vRt2
```
